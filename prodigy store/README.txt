Prodigy Store Versión 1.0

Bienvenido a una tienda digital moderna e interactiva, enfocada en productos electrónicos como auriculares, consolas, cargadores, smartwatchs, periféricos y más.

________________________________________________________


¿Qué tecnologías se usaron?

- HTML semántico
- CSS3 con Flexbox y Grid
- JavaScript moderno (`fetch`, DOM, `localStorage`)
- Diseño responsive (mobile-first)
- Font Awesome + Google Fonts
- Chatbot integrado (Tawk.to)
- Modo Oscuro

________________________________________________________

Funcionalidades

- Catálogo dinámico de productos
- Carrito de compras (con almacenamiento en `localStorage`)
- Buscador de productos en tiempo real
- Filtro por categoría
- Modo Oscuro/Día con toggle
- Chat en vivo con bot
- Formulario de contacto conectado a Formspree
- Código modular y ordenado
